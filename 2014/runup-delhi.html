Run-up Delhi

2014 September 06, 10:30 AM - 03:30 PM

At this JSFoo run up in Delhi, explore the various realms of applications of JS, ranging from concepts and open projects to platforms and winding up with best practices

Free event - RSVP here

Schedule

Time Sessions
10:15 AM - 10:30 AM Registrations & Introductions
10:30 AM - 11:00 AM

Talk: A new approach to scientific computing using Javascript

Rudi MK

In this talk, we're going to look at the viability of using Javascript as the main tool in one's arsenal, for mathematical and scientific computing. We're going to explore some of the major libraries available in this field, and we're going to cook up sample scenarios that use Javascript on the server for scientific computation, on the client for visualization and across embedded and robotic hardware, which could be a prelude to building more intelligent machines.
11:00 AM - 01:00 PM

Workshop: Learning JS concepts/techniques by implementing jQuery

Kushagra Gour

This is a “Re-implement and Learn” workshop. The aim of this workshop is to have a list of some concepts and common techniques used in JavaScript and then learn them by implementing parts of an existing library. For this session, we’ll be implementing features of jQuery to understand JavaScript more closely.

    Understanding the concepts implemented in this workshop will enable you to write code which is:
  1. More Clean
  2. More Manageable
  3. More Testable
01:00 PM - 02:00 PM Lunch hosted by Wingify
02:00 PM - 02:30 PM

Talk: Understanding WebRTC: A Whirlwind Tour

Shwetank Dixit

This talk will explain to you what WebRTC is, why is it important, and also guide you through the often misunderstood signalling process for forming real-time, p2p communications on the Web.
02:30 PM - 03:00 PM

Talk: Managing API Resources and Their Relationships on the Front-end

Himanshu Kapoor

This talk would go over the challenges, we at Wingify, faced while making our Angular.js based Single Page App – Visual Website Optimizer. One of the core challenges is efficiently communicating with a back-end service. Finally, the talk would conclude with the solutions we came up with to tackle these problems. It would also detail the problems faced with the seemingly trivial task of using Angular.js, to get and post data from a back-end API.
03:00 PM - 03:30 PM

Talk: Ten Useful Javascript Tips & Best Practices

Ankit Rastogi

In this session, we will discuss about Ten Javascript techniques which you can include in your applications along with some AngularJs tips and best practices as a bonus. These tips and best practices are accompanied by examples & will cover script loading, design pattern, performance optimization and other areas.
